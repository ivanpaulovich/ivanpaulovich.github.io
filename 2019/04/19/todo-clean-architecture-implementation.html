todo: My new Clean Architecture implementation

Posted by Ivan Paulovich on April 19, 2019 · 116 1 min read

I released a dotnet tool to manage tasks via the comnand line. The tasks are stored in a secret gist in your own GitHub account. Checkout a few commands:

Todo running in the terminal.

If you have your machine up to date with the latest .NET Core SDK and you are interest to see todo in action please run:

$ dotnet tool install -g todo

After that you need to grant access gist permissions to the todo tool.

In case you are running in a OSX checkout the issue with zsh: command not found: todo. Ask me about it!
